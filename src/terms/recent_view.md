# Recent View (RECENT_A)

## Definition
Recent View is a computed perspective that surfaces the most relevant causal state from an append-only event history.

## In Spiral
- Preserves explainability.
- Avoids destructive updates.
- Anchors interpretation in traceable events.

## Structure
Derived from trace graphs and fragment metadata.

## Notes
Recent View is not storage. It is interpretation.
