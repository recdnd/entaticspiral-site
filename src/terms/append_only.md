# Append-only

## Definition
Append-only describes a storage and recording principle where existing records are never modified or deleted.

## In Spiral
- Guarantees historical integrity.
- Enables reproducible traces.
- Shifts correction from mutation to extension.

## Structure
All new information is added as new fragments.

## Notes
Append-only does not prevent interpretation changes.
